This lib contains functions that have to do with audio.

Notes:

Like with the particles, the class Data requires certain amount of stuff to keep the sync with the video.

  1. The method “read” must be called one time for each frame; no more, no less.
  2. The frames must be accessed in order.
  3. Frames should not be skipped.

Lastly, the use of Data required to have installed ffmpeg and be available (meaning, to be in the path or the folder of kafx.)
