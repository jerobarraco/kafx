libs.asslib¶

This module defines the necessary things to load an ASS file, and the objects that allow to draw text with cairo

class libs.asslib.Ass(file, max)¶

This class parses the .ass file, luckily you won’t need to use it

LoadFromFile(file, max)¶

Loading a ass file

libs.asslib.TimeToMS(time)¶

Converts a string from type ‘0:00:00.00’ from ASS to miliseconds in integer

class libs.asslib.cDialogue(dialogue, styles, max_effect=0)¶

A Dialogue represents a text line, This takes the text, each Syllable of the text and it’s karaoke times. This object is the most complex one, almost imposible for you to create, better use cSyllable or directly extra.cVector

Chain(function, duration=None)¶

Allows to chain the Syllables to a animationn

Parameters:
  • function (method) – function to call for each syllable and the progress
  • duraton – duration of the animation of each syllable If not specified, a duration that animates one syllable per time will be used.
